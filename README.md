# FusionGEO Analyzer 🌐🤖
**Generative Engine Optimization (GEO) Audit & Auto-Fix Platform.**

FusionGEO Analyzerは、WebサイトがChatGPT、Perplexity、Claude等のAIエージェントに「ソースとして採用されるか」を解析し、最適化するためのエンジニアリング・エンジンです。

## 🧠 4つのコア・アプローチ
- **[人] Trust Graph:** 実名・職歴に基づいた信頼性の担保（LinkedIn連携）
- **[技術] Extractability:** AIが情報を抽出するための構造的最適化（llms.txt / Chunking）
- **[価値] Information Gain:** 学習済みデータにはない「情報の差分」の提供
- **[拡張] Co-occurrence:** Web全体でのブランド共起性とマルチモーダル対応

## 🛠️ 機能
- **BYOK (Bring Your Own Key):** ユーザー自身のAPIキー（OpenAI等）を使用してセキュアに解析。
- **20項目GEO監査:** 技術、信頼、価値、拡張の全方位チェック。
- **Auto-Fix提案:** `llms.txt`の自動生成やAI向けリライト案の提示。

## 🚀 クイックスタート
1. リポジトリをクローン
2. `pip install -r requirements.txt`
3. `streamlit run app.py`
